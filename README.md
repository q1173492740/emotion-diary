# 情绪日记 - AI情绪陪伴与管理系统

一款基于 Web 的情绪管理应用，提供日记记录、AI 情绪陪伴、呼吸练习和情绪数据分析功能。

## ✨ 功能特性

### 📔 情绪日记
- 心情选择（开心、平静、难过、焦虑、愤怒等）
- 日记撰写与标签管理
- 历史记录搜索与筛选
- 数据本地存储（LocalStorage）

### 🤖 AI 情绪陪伴
- 智能识别 15+ 情绪场景
- 共情式回应，避免空洞安慰
- 任务型对话（优点盘点、自信练习、思路梳理）
- 个性化推荐（音乐、甜食等）
- 产品功能联动（治愈空间、写日记）

### 🌿 治愈空间
- 可视化 478 呼吸法练习
- Web Speech API 语音引导
- 多端兼容

### 📊 情绪分析
- 7 天情绪趋势图（Chart.js）
- 情绪分布统计
- 智能洞察与预测

## 🚀 快速开始

直接用浏览器打开 `EmotionDiary.html` 即可使用。

## 🛠️ 技术栈

- **前端**: HTML5 / CSS3 / JavaScript (ES6+)
- **数据可视化**: Chart.js
- **语音功能**: Web Speech API
- **数据存储**: LocalStorage
- **开发方式**: Vibe Coding（AI 辅助开发）
