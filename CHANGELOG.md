# 情绪日记 - 更新日志

## v2.0 - AI情绪陪伴系统全面优化

### 🎯 核心优化

#### 1. 情绪优先级判断规则
- 重构情绪识别逻辑，采用「情绪基调优先 + 话题中性」原则
- 积极情绪词优先匹配，避免话题词（如简历、工作）默认导向消极
- 支持上下文情绪继承，保持对话情绪一致性

#### 2. 新增15+情绪场景识别
| 场景 | 关键词 |
|-----|--------|
| 求职面试 | 简历、面试、找工作、投、offer、求职 |
| 工作不顺 | 同事、领导、老板、上班、工作、加班、离职 |
| 人际关系 | 朋友、家人、吵架、矛盾、误会、冷战 |
| 学习考试 | 考试、学习、复习、挂科、成绩、考研 |
| 感情问题 | 分手、恋爱、喜欢的人、对象、感情 |

| 开心 | 开心、高兴、快乐、兴奋、好棒、太好了 |

| 迷茫 | 迷茫、不知道、方向、未来、人生意义 |

#### 3. 任务型对话流程（多轮对话承接）

**优点盘点（findStrengths任务）**
- 支持启动、承接、继续
- 用户说"要/好/可以"正确启动流程
- 用户说出优点后肯定并鼓励继续

**自信练习（confidencePractice任务）**
- 分步骤引导（自我肯定 → 小事成就 → 优点发现）
- 支持"挺好的/不错/好/好的"等正面回应承接
- 支持"还没/没有/想不出来/不知道"等困难回应，降低难度继续引导
- 支持"有/有啊/有的/有过"等发现回应，继续深入

#### 4. 回复质量优化
- 三层结构：共情 → 具体方法 → 温和引导
- 避免空洞安慰，每个场景都有可落地的具体建议
- 避免盘问式提问，改用开放式情绪引导
- 针对"你已经做得很好了"等空泛夸赞，改为基于具体行为的肯定

#### 5. 产品功能联动
- 深呼吸回复联动「治愈空间」模块（可点击跳转）
- 写下来/梳理思路联动「写日记」功能
- 动态话题标签：根据场景切换底部快捷按钮

#### 6. 个性化推荐
- 音乐推荐支持：二次元、摇滚、古典、民谣等类型
- 甜食推荐围绕甜点展开（不再推荐正餐）

### 🐛 修复的问题

- 修复快捷按钮「聊聊最近」发出的内容像AI提问的问题
- 修复「想做深呼吸练习」被误判为其他场景
- 修复「二次元」音乐类型无法识别
- 修复「因为你」指向AI的负面情绪无回应
- 修复自信练习流程中「还没/有/好」等回应无法承接
- 修复「帮我梳理一下思路」只有被动倾听
- 修复「能给我打打气吗」只有被动倾听
- 修复「如何提升自信」只有被动倾听
- 修复「接地法」只有被动倾听
- 修复「等HR回复」只有空洞安慰
- 修复「我想聊聊为什么焦虑」只有鸡汤
- 修复投简历默认导向焦虑（忽略积极情绪）
- 修复上下文追问识别失效（"吃什么"/"什么歌"等）
- 修复「我陪着你」快捷按钮空洞无意义（改为「写下来」）
